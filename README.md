An end-to-end analysis of **60,000+ IPL deliveries and match records** using:

- **SQL** for extracting match-level & player-level insights  
- **Python (Jupyter)** for data validation & EDA  
- **Tableau Public** for interactive visualization  

This project answers key analytical questions such as:

- Which teams consistently performed best across seasons?  
- Who are the top run-scorers & wicket-takers?  
- Does winning the toss affect match results?  
- Which bowlers maintain the lowest economy rates?
